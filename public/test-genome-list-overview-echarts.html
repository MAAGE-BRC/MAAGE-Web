GenomeListOverview with Apache ECharts Integration

Integration Instructions

To use the new ECharts-based GenomeListOverview widget in your application:

  1. Replace 'p3/widget/GenomeListOverview' with 'p3/widget/GenomeListOverviewECharts' in your require statements
  2. The widget API remains the same - no code changes needed
  3. ECharts is already loaded globally via the head.ejs template
  4. The MAAGE theme is automatically applied to all charts

Explore Surveillance Data

This step-by-step guide helps you explore genomic surveillance data through interactive dashboards. Review preconfigured national, regional, and state surveillance dashboards, or create custom dashboards from genome search results to monitor pathogen circulation, identify emerging clusters, and analyze trends across geography, time, lineage, and other metadata relevant to your investigation.

Step 1. Choose a Surveillance View

Start by selecting how you want to explore surveillance data.

Public Dashboards

Provide ready-to-use national, regional, and state views for priority pathogens. Use these to get a quick overview of sequenced pathogens without needing to define search criteria.

Custom Dashboards

Allow you to create your own custom surveillance view using a variety of genome search criteria, such as organism, location, collection date, host, source, lineage, serotype, MLST/cgMLST cluster, or other metadata. You can also customize the dashboard layout to highlight desired metadata attributes.

Step 2. Explore Public Surveillance Dashboards

Explore preconfigured national, regional, and state surveillance dashboards to monitor pathogen circulation, identify emerging clusters, and examine trends across geography, time, lineage, and other epidemiological metadata.

Available dashboard types

  • • National Genomic Surveillance
  • • Regional Genomic Surveillance
  • • State Genomic Surveillance
  • • Priority pathogen dashboards
  • • Outbreak- or cluster-focused dashboards

Use this step to review

  • • Genome counts over time
  • • Geographic distribution
  • • Public isolate clusters
  • • Lineages, serotypes, or sequence types
  • • Available metadata summaries

Step 3. Search Genomes to Define a Custom Dataset

If the preconfigured dashboards do not match your investigation needs, use Genome Search to define a custom surveillance dataset.

Select genomes by filtering on:

  • • Organism or pathogen group
  • • Country, region, state, or local jurisdiction
  • • Collection date range
  • • Host
  • • Isolation source
  • • Lineage, clade, serotype, MLST, or HierCC cluster
  • • AMR or virulence markers
  • • Public or private genome status
  • • User-defined metadata fields

This creates a focused genome set for surveillance, outbreak monitoring, or follow-up analysis.

Step 4. Create a Custom Surveillance Dashboard

Save your selected genome search as a custom dashboard. MAAGE uses the selected genomes and metadata to generate an interactive dashboard for your investigation.

Your custom dashboard can summarize

  • • Temporal trends
  • • Geographic spread
  • • Cluster membership
  • • Metadata distributions
  • • Lineage or strain composition
  • • AMR and virulence patterns
  • • Related public and private isolates

Step 5. Review and Refine Results

Use the dashboard to explore patterns, identify unusual clusters, and refine your search criteria as needed. You can adjust filters, narrow the date range, focus on specific locations, add or remove metadata fields, or select particular clusters for deeper investigation.

Use this step to answer

  • • What is circulating?
  • • Where is it circulating?
  • • Has activity changed over time?
  • • Are new clusters emerging?
  • • Are related isolates appearing in multiple locations?
  • • Which genomes should be selected for further analysis?

Step 6. Save, Share, or Launch Downstream Analysis

Once your dashboard captures the surveillance view you need, save it to My Dashboards for future monitoring.

From the dashboard, you can continue into downstream MAAGE workflows when needed, including:

  • • Bacterial Investigation Workflow
  • • Viral Outbreak Investigation Workflow
  • • Whole Genome SNP Analysis
  • • Phylogenetic Analysis
  • • Comparative genome analysis
